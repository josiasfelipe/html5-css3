Imagen dinâmica

Tenta abrir esse site em vários dispositivos diferentes ou simplesmente aumente e diminua o tamanho do seu navegador.

(Tag usada: <picture>, dentro desta tag se utiliza a tag <img> para definir a imagem padrão, sendo a menor aqui definida como tal. Logo acima da tag <img> se coloca as outras imagens maiores subsequentes do menor para o maior, de baixo para cima, utilizando a tag <source> com media-type para definir tais imagens).
