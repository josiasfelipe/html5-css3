Usando links externos

Podemos usar links externos através da tag 'a'. É importando destacar links externos em outras abas do navegador usando as referências 'tagert="_blank"' e 'rel="external"' em conjunto.

Por exemplo: você pode acessar o YouTube clicando aqui.

Usando links internos

Podemos usar a mesma tag 'a' para conectar arquivos html internos, enriquecendo a nevagação do site.
Se os arquivos estiverem na mesma pasta, basta referencialos em href="", se estiverem em uma subpasta, ficar atendo aos caminhos do diretório descrito.

Exemplo: Você pode acessar a segunda página aqui.
(Arquivo html na mesma página do index principal)

Ou pode acessar um arquivo html em uma subpasta na página 03.

Usando links para download

Podemos usar ainda a mesma tag 'a' para criar links para downloads, usando referências na tag: download="arquivo.zip" e a type="application/zip" respectivamente. Importante referenciar bem o href, atenção para o diretório (sendo indicado uma subpasta para hospedar o arquivo a ser baixado) e ao formato deste arquivo.
