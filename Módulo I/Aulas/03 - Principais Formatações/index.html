Principais formatações

Negrito / Destaque

Nesta frase, temos um termo em negrito usando a tag 'b' (não semâmtico).

Nesta frase, temos um termo em dastaque usando a tag 'strong' (semântico).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag 'i' (não semântico)

Nesta frase, temos um termo em ênfase usando a tag 'em' (semâmtico)

Texto marcado

Podemos criar também um texto marcado usando a tag 'mark'

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo. (a tag 'big' usada aqui para texto grande está em vias de ser descontinuada; a tag 'small' permanece sendo usada e de forma semântica)

Texto excluído

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Usando a tag 'del'.

Texto Sublinhado/Inserido

Podemos sublinhar um texto usando a tag 'u' (não semântica).

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Usando a tag 'ins'(semâmtico).

Texto sobrescrito

Quando queremos elevar o nível do caractere, como a potência neste exempo: x2 (usando a tag 'sup').

Texto subscrito

Quando queremos descer o nível do caractere, como o número de hidrogênio na fórmula da água H2O (usando a tag sub).
